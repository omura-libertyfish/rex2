`<<識別子`行から`識別子`行の直前までを文字列として扱います。このような表現方法をヒアドキュメントといいます。
問題では、ヒアドキュメントにインデントを加えています。
`<<識別子`とした際は、インデントを加えることはできず、その場合は、エラーになります。

ヒアドキュメントにインデントを加える場合は、`<<-識別子`とします。

```ruby
s = <<-EOF
    Hello,
    Ruby
    EOF
p s
# <実行結果>
#     Hello,\n    Ruby\n
```

インデントを取り除くためには、Ruby 2.3から追加された`<<~識別子`を用います。
