左辺の`*`は右辺の余った要素を無視します

```ruby
例:
*, var1, var2 = 1, 2, 3
p var1 # => 2
p var2 # => 3
```

メソッドの仮引数に`*`を付けるとメソッドが受け取った実引数の余剰分を全てを配列として受け取ります。これを可変長引数と呼びます。  

**例1**

```ruby
def func(a, *b)
  p "#{a} / #{b}"
end

func(1, 2, 3, 4) # => "1 / [2, 3, 4]"
```

実引数に`*`を付けることで引数に配列を展開します。

```ruby
def func(a, b, c, d)
  p "#{a} / #{b} / #{c} / #{d}"
end

arr = [1, 2, 3, 4]
func(*arr) # => "1 / 2 / 3 / 4"
```
