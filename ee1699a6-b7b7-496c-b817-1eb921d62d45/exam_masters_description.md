`&`, `&&`, `|`, `||`演算子それぞれの動きの違いについて問題です。

`&`と`&&`の違いについて
`&`は共通する要素を出力する
```ruby
a = [1, 2, 3]
b = [1, 3, 5]
a & b #=> [1, 3]
```

`&&`は a と b がどちらも真のとき、最後に評価された b を出力する
```ruby
a && b #=> [1, 3, 5]
```

`a = nil`の場合右辺の処理は行われない
```ruby
a && b #=> nil
```

`|`と`||`の違いについて
`|`は重複している要素をまとめて出力する
```ruby
a = [1, 2, 3]
b = [1, 3, 5]
a | b #=> [1, 2, 3, 5]
```

`||`は a と b がどちらかが真のとき、最初に評価された a を出力する
```ruby
a || b #=> [1, 2, 3]
```

`a = nil`の場合右辺の処理が行われる
```ruby
a || b #=> [1, 3, 5]
```
